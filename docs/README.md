# Platforma transmisji online z ceremonii pogrzebowej — SRS

Repozytorium zawiera dokument Specyfikacji Wymagań Oprogramowania (SRS) dla systemu umożliwiającego prywatną transmisję online z ceremonii pogrzebowej.

## Cel repo
- Stworzenie jednoznacznego SRS (wspólny kontrakt zespołu i interesariuszy).
- Utrzymanie historii pracy w Git (PR, review, regularne commity).

## Jak pracujemy
- Każda sekcja dokumentu ma właściciela.
- Zmiany robimy przez Pull Requesty z review.
- Wymagania numerujemy jako: WF-XXX-YY oraz WNF-XXX-YY.

## Struktura dokumentu
Pliki w katalogu `docs/` odpowiadają rozdziałom SRS i są składane do `00_SRS_main.md`.
